  煅火行枢是一款基于架空世界观的开放世界探索的物理游戏，以东方机关术与古典物理学为核心的单机解谜冒险游戏。玩家将扮演少年工匠方源，手持祖传奇书《天工开物》，探索大世界收集材料，踏入千年墨家学派废墟，挑战引发地震的罪魁祸首赤枢。通过破解物理谜题、重构古籍残卷、锻造克敌兵器，揭开墨家工艺湮灭的真相，用物理法则终结这场灾难。

  
  “从磁石悬针到火龙焚天，每一件道具都是物理法则的具象，每一卷古籍皆是文明跃迁的碑铭。”游戏内的所有道具和书本均有对应的中国古代的物理古籍考究。

  
  古籍系统详解
  《驭火篇》（火药本源）
  记载唐代孙思邈《丹经内伏硫磺法》与清虚子秘方，揭示火药“三英汇聚”原理（硝引风、硫掀天、炭腾焰）
  警告火药失控之险：“若猛兽出笼，火光百里，震裂四壁”

  
  《司南篇》（磁力奥秘）
  引自沈括《梦溪笔谈》，详述磁针“缕悬法”与地磁偏角现象
  司南制作需“以磁石磨针锋，蜡缀独茧缕”，成就航海神器

  
  《冶铁篇》（铸剑真谛）
  传承《考工记》铸剑三阶：观“黑浊→黄白→青气”判定火候
  强调淬火定生死：“急速冷却成锋，磨砺去瑕为刃”

  
  《机关篇》（墨家机巧）
  摘录《墨子》攻城器械论，主张“器巧胜勇，云梯飞石皆为道”
  暗藏齿轮联动奥妙。

  
  《焰火剑篇》（爆炎武学）
  源自北宋《武经总要》，剑招“火龙腾空”需内力引燃剑腹火药珠
  致命风险：剑气与爆破力需严格掌握精髓，才能使用华丽的连击。

  
  《天工开物》（文明总纲）
  明末宋应星原著精髓：“五谷贵于金玉，器物成于格物”
  神秘书灵：玩家可与接入Deepseek的书灵询问游戏内的物理问题。
